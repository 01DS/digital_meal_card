# 🍽️ Digital Meal Card System

## 📖 Overview
The Digital Meal Card System is a full-stack web application developed as a university project. It provides a digital solution for managing meal cards, making the process more efficient, secure, and easier to manage.

## ✨ Features
- User authentication
- Digital meal card management
- Database integration
- Responsive web interface
- Secure data storage

## 🛠️ Technologies Used
- Java
- Node.js
- HTML5
- CSS3
- JavaScript
- MySQL

## 🚀 Getting Started
1. Clone this repository.
2. Install the required dependencies.
3. Configure the MySQL database.
4. Start the application.
5. Open the application in your browser.

## 🎓 About
This project was developed as part of a university software development course to demonstrate full-stack development, database integration, and web application design.
